# PyLAR
## Python utilities for the LAR model
### Paper figures

This directory contains data, scripts and notebooks required to create the figures of the papers:

1. **LAR-SMZG-HESS2024**:  Juan F. Salazar, Rubén D. Molina, Jorge I. Zuluaga, and Jesus D. Gomez-Velez (2024), **Wetting and drying trends in the land–atmosphere reservoir of large basins around the world**, [Hydrology and Earth System Sciences, in publication (2024)](https://hess.copernicus.org/preprints/hess-2023-172/), [doi.org/10.5194/hess-2023-172](https://doi.org/10.5194/hess-2023-172).
